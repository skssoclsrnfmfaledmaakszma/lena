* 수업 전
1) java, eclipse 다운로드
www.oracle.co.kr
www.eclipse.org
2) C:\ 하위에 본인폴더 생성
3) eclipse 구동 시 workspace를 C:\본인폴더로 설정
4) eclipse 인코딩 설정
File > Properties > Resource > 하단 Text file encoding > Other > UTF-8로 변경
5) 불러오기
USB폴더를 C:\본인폴더 하위에 복사 후
eclipse 좌측 project explorer 우클릭 > Import > Import 클릭,
General > Projects from Folder or Achive > Next버튼
우측 상단 Directory > C:\본인폴더\복사폴더 > 폴더선택 버튼
